## 🚀 Instalación y Uso

### Prerrequisitos
- Node.js 18+ 
- npm o yarn

### Clonar el repositorio
```bash
git clone https://github.com/OpceanAI/opceanai-portfolio.git
cd opceanai-portfolio
```

### Instalar dependencias
```bash
npm install
```

### Ejecutar en desarrollo
```bash
npm run dev
```

### Construir para producción
```bash
npm run build
```
